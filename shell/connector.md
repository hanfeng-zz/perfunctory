| 连接符  | 作用说明                                         |                       |                           |
| ---- | -------------------------------------------- | --------------------- | ------------------------- |
| `;`  | 顺序执行前后命令，不管前面的命令是否成功                         |                       |                           |
| `&&` | 前一个命令成功（退出码为 0）时，才执行后一个命令                    |                       |                           |
| \`   |                                              | \`                    | 前一个命令失败（退出码非 0）时，才执行后一个命令 |
| `&`  | 将命令放到后台执行（异步执行）                              |                       |                           |
| \`   | \`（管道）                                       | 将前一个命令的输出传递给后一个命令作为输入 |                           |
| `()` | 在子 Shell 中执行一组命令，互不影响当前 Shell 环境             |                       |                           |
| `{}` | 在当前 Shell 中按块执行命令，命令之间需用 `;` 分隔（末尾需加空格或 `;`） |                       |                           |
